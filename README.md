# 🚀 Placement Training Website

A full-stack web application designed to help students prepare for campus placements by providing learning resources, mock tests, coding practice, and AI-powered assistance.

---

## 📌 Features

* 👤 User Authentication (Login/Register)
* 📚 Course & Study Materials (Aptitude + Technical)
* 📝 Mock Tests with Auto Evaluation
* 💻 Coding Practice Platform
* 📄 Resume Builder
* 🤖 AI Chatbot for Doubts & Interview Prep
* 📊 Performance Dashboard
* 🏢 Company-wise Preparation Section

---

## 🧱 Tech Stack

**Frontend:**

* React.js / Next.js
* HTML, CSS, JavaScript

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB / MySQL

**AI Integration:**

* OpenAI API

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/placement-training-website.git
cd placement-training-website
```

### 2️⃣ Install dependencies

**Backend:**

```
cd backend
npm install
```

**Frontend:**

```
cd frontend
npm install
```

---

### 3️⃣ Configure Environment Variables

Create a `.env` file in backend folder:

```
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
OPENAI_API_KEY=your_key
```

---

### 4️⃣ Run the project

**Backend:**

```
npm start
```

**Frontend:**

```
npm start
```
